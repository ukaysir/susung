## 1. 완료 조건

다음을 모두 갖추면 제출 준비가 완료된 것으로 본다.

- 실제 수성구 후보시설 최소 1건의 위치·사진·현재 기능 확인
- 후보시설이 수성구 소유 또는 관리 대상인지 확인
- 법정·안전·교통약자 시설이 아니라는 1차 확인
- 기존 도시비우기와 다른 ‘30일 가역적 빈자리 실험’ 절차 명시
- 실험 전후를 비교할 관찰지표 명시
- 예산을 확정값이 아닌 확인 가능한 비용 항목으로 제시
- 공고문 제외사유와 제출형식 검수
- 최종 HWPX 파일 열람 및 레이아웃 확인

## 4. 제안서 필드별 설계

### 제안 제목

- 확정안: **없애기 전에 없어져 본다 — 수성구 「공공시설 빈자리 실험」**
- 기능: 역설적 관점과 핵심 작동원리를 동시에 전달한다.

### 개요

- 첫 문장: 오래되거나 중복된 공공시설물을 바로 철거하지 않고, 가역적인 빈자리를 먼저 실험한다.
- 포함할 내용: 대상, 30일 기본기간, 전후 비교, 네 가지 결정 결과.
- 금지: 전국 최초, 예산 절감 확정, 모든 공공시설 적용.

### 현황 및 문제점

- 실제 후보시설 사진과 관찰 사실을 먼저 제시한다.
- ‘시설물이 많다’가 아니라 ‘없는 상태를 시험하지 못해 존치와 철거 판단 근거가 부족하다’는 의사결정 문제로 연결한다.
- 수성구 전체 시설물 수나 유지비는 공식 자료를 확보한 경우에만 사용한다.

### 개선방안

- 후보 제안 → 적격성 검토 → 기준선 기록 → 잠정 퇴역 → 전후 비교 → 결정·공개 순으로 쓴다.
- 안전·법정·교통약자 시설 제외와 즉시복원 조건을 가까이 배치한다.
- 앱 제작이나 주민 인기투표를 핵심 절차로 넣지 않는다.

### 기대효과

- 산출: 후보 검토표, 전후 측정자료, 시설별 결정 기록.
- 결과: 보행공간 변화와 시설 필요성을 실제 반응으로 판단할 수 있음.
- 장기 효과: 불필요한 시설의 누적과 유지관리 부담을 줄일 가능성 검증.
- 근거 없이 금액·비율 목표를 만들지 않는다.

### 조치사항

- 총괄부서와 시설 관리부서의 역할 정리.
- 실험 대상 선정 기준과 제외 기준 마련.
- 가림·운반·보관·복원 절차 마련.
- 실험 전후 기록표와 결정 공개서식 마련.
- 법률·안전 검토가 필요한 대상은 실험에서 제외.
