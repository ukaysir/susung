# 확정 아이디어 — 수성구 공공시설 빈자리 실험

- 확정일: 2026-07-17
- 적용 공모: 2026년 수성구 제안 공모
- 상태: 아이디어 확정 / 현장 근거 수집 전

## 1. 제안명

**없애기 전에 없어져 본다 — 수성구 「공공시설 빈자리 실험」**

## 2. 한 문장 제안

기능을 잃었거나 중복됐다고 제안된 수성구 소유의 비안전·비법정 공공시설물을 곧바로 철거하지 않고 기본 30일 동안 가리거나 임시 이동한 뒤, 주민 불편과 보행환경·유지관리 변화를 비교하여 복원·통합·이전·철거를 결정한다.

## 3. 제안 개요

공공공간에는 설치 목적이 사라졌거나 기능이 중복됐지만, 철거에 따른 책임과 불확실성 때문에 계속 유지되는 시설물이 있을 수 있다. 기존 도시정비는 담당 부서가 필요성을 판단한 뒤 바로 철거하는 방식이 일반적이다.

「공공시설 빈자리 실험」은 최종 결정을 내리기 전에 되돌릴 수 있는 빈자리를 먼저 만든다. 주민이나 현장 공무원이 후보를 제안하면 소유권·법적 의무·안전·교통약자 영향을 확인하고, 적격 대상만 가림막을 씌우거나 별도 장소에 임시 보관한다. 실험 전후의 문의·불편, 보행공간, 유지관리 업무 변화를 기록한 뒤 복원·통합·이전·철거 중 하나를 결정하고 이유를 공개한다.

핵심은 시설물을 많이 없애는 것이 아니다. **시설이 없는 상태를 먼저 시험해, 유지와 철거 중 어느 결정이 주민에게 더 나은지를 실제 반응으로 확인하는 것**이다.

## 4. 문제–개입–결과 구조

| 요소 | 확정 내용 |
| --- | --- |
| 대상 | 공공공간을 이용하는 주민과 시설 관리부서 |
| 상황 | 오래되거나 중복된 공공시설물이 계속 누적되는 상황 |
| 문제 | 바로 철거하기에는 위험하고, 그대로 두기에는 보행 방해와 관리 부담 여부를 확인하기 어려움 |
| 원인 | 실제로 없애본 상태의 이용 반응 없이 서류와 회의만으로 존치 여부를 판단함 |
| 개입 | 저위험 대상의 가역적 가림·임시 이동 실험 |
| 작동원리 | 실제 빈자리를 만들어 시설의 필요성과 제거 효과를 동시에 관찰함 |
| 직접 결과 | 복원·통합·이전·철거를 판단할 현장 근거 생성 |
| 기대 변화 | 불필요한 시설의 정비 가능성, 보행공간 개선 가능성, 유지관리 부담 감소 가능성 검증 |
| 측정 | 실험 전후 문의·불편, 유효 보행폭, 관리 횟수·시간·비용, 복원 필요 여부 |
| 실행 주체 | 수성구 총괄부서와 개별 시설 관리부서(정확한 담당부서 확인 필요) |
| 차별성 | 조사 후 바로 철거하지 않고, 되돌릴 수 있는 ‘부재 상태’를 먼저 실험함 |

## 5. 실험 대상과 제외 대상

### 실험 후보

- 내용이 중복되거나 오래된 관광·행정 안내판
- 행사가 끝난 뒤 남은 비규제 안내 지주
- 장기간 비어 있거나 기능이 중복된 공공 홍보게시판
- 용도를 잃은 공공 조형물의 비안전 부속시설
- 같은 장소에 반복 설치된 비규제 안내시설

위 목록은 가능한 유형의 예시이며, 수성구 내 실제 사례로 확인된 사실이 아니다. 제안서에는 현장조사로 확인한 후보만 사용한다.

### 원칙적 제외

- 교통·소방·방범·재난·생명안전 관련 시설
- 법령이나 상급기관 지침상 의무시설
- 장애인·아동·고령자 등 특정 이용자를 위한 편의시설
- 수성구가 소유하거나 관리하지 않는 시설
- 계절성 때문에 30일 관찰로 필요성을 판단하기 어려운 시설
- 임시 이동이나 가림 뒤 원상복구가 어려운 시설

## 6. 작동 절차

1. **후보 제안**: 주민 또는 현장 공무원이 정확한 위치, 현재 기능, 중복·불편 사유를 제출한다.
2. **적격성 검토**: 관리주체, 법적 의무, 안전기능, 교통약자 영향을 확인한다.
3. **기준선 기록**: 시설 상태, 이용 장면, 유효 보행폭, 유지관리 업무를 기록한다.
4. **잠정 퇴역**: 기본 30일 동안 비파괴 가림 또는 임시 이동을 실시한다.
5. **전후 비교**: 시설을 찾는 문의와 불편, 보행환경, 관리업무 변화를 기록한다. 위험이 확인되면 즉시 복원한다.
6. **결정·공개**: 복원·통합·이전·철거 중 하나를 결정하고 판단 이유를 공개한다.

## 7. 평가항목 대응

공고문에 확인된 평가항목은 창의성, 경제성, 실용성, 주민 편의성, 노력도이며 항목별 배점은 공개되지 않았다.

| 공식 평가항목 | 현재 강점 | 제출 전 보강할 근거 |
| --- | --- | --- |
| 창의성 | ‘철거’보다 먼저 ‘부재 상태’를 실험하는 의사결정 방식 | 기존 도시비우기·시설물 신고와 작동 절차 비교 |
| 경제성 | 신규 플랫폼이나 대규모 건설 없이 시범 가능 | 가림·운반·보관·조사에 필요한 비용 항목, 유지관리 부담 비교 |
| 실용성 | 수성구 소유의 저위험 대상부터 한정 실시 가능 | 실제 후보의 소유·관리부서와 원상복구 방식 |
| 주민 편의성 | 보행 방해와 시각적 혼잡을 현장에서 검증 | 주민 이용 장면, 유효 보행폭 등 실제 현장자료 |
| 노력도 | 안전 검토와 전후 측정 구조가 명확함 | 현장사진, 후보 비교표, 관계 부서 확인 기록 |

## 8. 선행제도 검증 결과

### 확인된 중복

- 수성구는 2022년 규제개혁위원회 조례를 개정해 담당 공무원이 규제의 존치 필요성을 설명하는 ‘규제입증책임제’ 근거를 마련했다. 따라서 ‘규제의 입증책임을 뒤집는다’는 표현과 제도는 본 아이디어의 차별점으로 사용하지 않는다.
- 부산시 등은 불필요한 공공시설물과 조형물을 정리하는 도시 비우기 사업을 추진했다. 따라서 단순 철거·정비도 차별점으로 사용하지 않는다.
- 해외 지자체에도 주민이 중복되거나 오래된 표지판의 철거를 요청하는 사례가 있다. 주민 신고창구 자체도 차별점으로 사용하지 않는다.

### 남길 차별점

**최종 철거 전에 시설이 없는 상태를 가역적으로 만들고, 전후 변화를 측정해 의사결정한다.**

공개 자료 검색에서 동일한 절차의 국내 제도를 확인하지 못했지만, 전수조사가 아니므로 ‘전국 최초’ 또는 ‘유일’이라고 주장하지 않는다.

## 9. 변경 금지선

다음 변경은 아이디어의 차별성을 없애므로 하지 않는다.

- 규제입증책임제를 새로 도입한다는 제안으로 변경
- 단순 시설물 신고·철거 사업으로 축소
- 주민투표만으로 안전·편의시설을 철거
- 앱·지도·인증제 개발을 핵심으로 추가
- 철거 개수 자체를 성과로 설정
- 근거 없이 ‘전국 최초’, ‘유일’, ‘예산 절감’을 단정
