# 확정 아이디어 — 수성구 공공시설 빈자리 실험

- 확정일: 2026-07-17
- 적용 공모: 2026년 수성구 제안 공모
- 상태: 아이디어·운영구조 확정 / 제출자 인적사항 입력 전

## 1. 제안명

**없애기 전에 없어져 본다 — 수성구 「공공시설 빈자리 실험」**

## 2. 한 문장 제안

주민이나 현장 공무원이 기능 중복·용도 상실 가능성이 있는 공공시설물을 제안하면, 수성구가 소유권과 법정 의무·안전·교통약자 영향을 먼저 검토하고 적격 시설만 기본 30일 동안 가리거나 임시 이동한 뒤 전후 변화를 비교하여 복원·통합·이전·철거를 결정한다.

## 3. 제안 개요

공공시설물의 존치 여부는 대개 시설이 있는 상태의 자료로만 판단한다. 철거 후 문제가 생길 우려가 있으면 그대로 두기 쉽고, 철거를 결정하면 실제 영향을 확인하기 전에 되돌리기 어려워진다. 문제의 핵심은 특정 시설이 불필요하다는 것이 아니라, **시설이 없는 상태를 안전하게 시험할 중간 단계가 없다는 것**이다.

「공공시설 빈자리 실험」은 최종 처분 전에 되돌릴 수 있는 부재 상태를 만든다. 주민이나 현장 공무원이 후보를 제안하고, 수성구 관리부서가 적격성을 확인한다. 적격 시설은 비파괴 가림이나 임시 이동 방식으로 기본 30일간 잠정 퇴역한다. 실험 전후의 문의·불편, 보행환경, 시설 기능, 관리업무 변화를 같은 기준으로 기록한다. 위험이나 필수 기능이 확인되면 즉시 복원하고, 실험 종료 후에는 복원·통합·이전·철거 중 하나를 근거와 함께 결정한다.

성과는 철거 개수가 아니다. **존치와 철거 사이에 검증 단계를 넣어, 어느 결정이 주민에게 더 나은지 설명할 수 있는 행정 근거를 만드는 것**이 성과다.

## 4. 문제–개입–결과 구조

| 요소 | 확정 내용 |
| --- | --- |
| 대상 | 공공공간을 이용하는 주민과 시설 관리부서 |
| 상황 | 기능 중복·용도 상실 가능성이 제기된 시설물의 존치 여부를 판단하는 상황 |
| 문제 | 그대로 두거나 철거하는 두 선택지만 있어, 시설 부재의 실제 영향을 확인하기 어려움 |
| 원인 | 최종 처분 전에 되돌릴 수 있는 현장검증 절차와 공통 측정표가 없음 |
| 개입 | 저위험·복원 가능 시설을 대상으로 한 기본 30일의 가역적 가림·임시 이동 |
| 작동원리 | 시설이 없는 상태를 실제로 만들고 동일 기준의 전후 자료를 확보함 |
| 직접 산출 | 후보 적격성 검토표, 전후 측정표, 복원 기록, 시설별 결정서 |
| 기대 변화 | 시설 필요성 판단의 설명력 향상, 보행·이용환경 개선 가능성 검증, 관리업무 조정 근거 확보 |
| 측정 | 문의·불편의 내용과 건수, 유효 보행폭, 이용행동, 관리 횟수·시간, 즉시복원 사유 |
| 실행 주체 | 총괄부서, 시설 관리부서, 필요한 경우 안전·교통·복지 관련 검토부서 |
| 차별성 | 조사 후 곧바로 철거하지 않고, 되돌릴 수 있는 ‘부재 상태’를 먼저 실험함 |

## 5. 적용 대상과 제외 대상

### 적용 가능한 유형

- 정보가 중복되거나 현행성이 의심되어 재검토가 필요한 비규제 안내시설
- 행사가 끝났거나 목적 변경 가능성이 제기된 임시·홍보 시설
- 게시 기능이 중복됐다고 제안된 공공 게시시설
- 독립적으로 가리거나 이동할 수 있는 공공 조형·편의시설의 비안전 부속물
- 같은 공간에서 유사 기능을 반복한다고 제안된 시설

이 목록은 제도의 적용 범위를 설명하는 유형이며, 수성구의 특정 시설이 불필요하다는 사실 주장이 아니다. 실제 후보 발굴과 적격성 판단은 제도 시행 단계에서 주민·현장 공무원·관리부서가 수행한다.

### 원칙적 제외

- 교통·소방·방범·재난·생명안전 기능을 담당하는 시설
- 법령·조례·상급기관 지침에 따른 의무시설
- 장애인·아동·고령자 등 특정 이용자의 접근과 편의를 보장하는 시설
- 수성구가 소유하거나 관리하지 않는 시설
- 계절·행사 주기 때문에 기본 실험기간으로 기능을 판단하기 어려운 시설
- 가림·이동이 새로운 위험을 만들거나 당일 원상복구가 어려운 시설
- 문화·역사·기념적 보존 여부를 별도로 판단해야 하는 시설

## 6. 작동 절차

1. **후보 제안**: 주민 또는 현장 공무원이 위치, 현재 기능, 재검토 사유와 관찰 사실을 제출한다.
2. **적격성 검토**: 관리부서가 소유·관리권, 법정 의무, 안전기능, 교통약자·소수 이용자 영향, 복원 가능성을 확인한다.
3. **실험계획 승인**: 임시 조치 방법, 실험기간, 측정항목, 대체안내, 즉시복원 담당자를 정한다.
4. **기준선 기록**: 실험 전 동일 요일·시간 조건에서 시설 이용과 문의, 보행환경, 관리업무를 기록한다.
5. **잠정 퇴역**: 시설을 훼손하지 않는 가림 또는 임시 이동으로 기본 30일간 부재 상태를 만든다.
6. **전후 비교**: 같은 조건으로 문의·불편, 이용행동, 보행환경과 관리업무 변화를 기록한다.
7. **결정·공개**: 복원·통합·이전·철거 중 하나를 결정하고, 측정 결과와 판단 이유를 공개한다.

## 7. 즉시복원 기준

다음 중 하나가 확인되면 30일을 채우지 않고 원상복구한다.

- 보행·교통·방재·야간 시야 등 새로운 안전위험이 발생함
- 검토에서 놓친 법정 의무나 필수 안내기능이 확인됨
- 장애인·아동·고령자 등 특정 이용자의 접근에 실질적인 지장이 발생함
- 대체수단으로 해결할 수 없는 기능 상실이 확인됨
- 가림·보관 장치의 훼손이나 시설 손상 우려가 발생함

복원은 실험 실패가 아니다. 해당 시설의 필요 기능을 확인한 결과로 기록한다.

## 8. 최종 결정 기준

| 관찰 결과 | 기본 결정 | 판단 취지 |
| --- | --- | --- |
| 필수 기능 또는 복원 사유가 확인됨 | 복원 | 시설의 필요성을 현장자료로 확인 |
| 기능은 필요하지만 위치·형태·정보가 중복됨 | 통합 또는 이전 | 필요한 기능을 유지하면서 공간 부담 조정 |
| 기능 상실이 관찰되지 않고 공간·관리 측면의 개선이 확인됨 | 철거 검토 | 별도 행정·법률 절차를 거쳐 최종 처분 |
| 자료가 부족하거나 계절·행사 영향이 큼 | 판단 유보 | 기간·측정조건을 보완한 뒤 재검토 |

주민 의견의 숫자만으로 결정하지 않는다. 관리부서의 기능 검토, 취약 이용자 영향, 안전자료와 현장측정을 함께 본다.

## 9. 공식 평가항목 대응

공고문에 확인된 평가항목은 창의성, 경제성, 실용성, 주민 편익 증진도, 노력도이며 항목별 배점은 공개되지 않았다.

| 공식 평가항목 | 판단 근거 |
| --- | --- |
| 창의성 | 존치·철거의 이분법 사이에 ‘가역적 부재 실험’을 새로운 행정 단계로 둠 |
| 경제성 | 신규 플랫폼 없이 기존 제안창구와 관리부서 업무로 시범 가능하며, 비용은 가림·운반·보관·복원·측정 항목으로 제한함 |
| 실용성 | 대상·제외 기준, 일곱 단계 절차, 즉시복원 기준, 네 가지 최종 결정이 문서화됨 |
| 주민 편익 증진도 | 주민이 후보를 제안하고, 실제 이용불편·보행환경·취약 이용자 영향을 결정자료로 사용함 |
| 노력도 | 후보 제안서, 적격성 체크리스트, 전후 측정표, 복원 기록서, 최종 결정서까지 운영서식을 설계함 |

## 10. 선행제도 검증과 차별점

- 수성구에는 담당 공무원이 규제의 존치 필요성을 설명하는 규제입증책임제의 근거가 있으므로, ‘입증책임 전환’ 자체를 차별점으로 주장하지 않는다.
- 다른 지자체에는 불필요한 공공시설물을 정리하는 도시 비우기 사업이 있으므로, 단순 정비·철거를 차별점으로 주장하지 않는다.
- 주민이 중복·노후 시설을 신고하는 창구도 존재하므로, 주민 제안 접수 자체를 차별점으로 주장하지 않는다.

본 제안이 남기는 차이는 **최종 철거 전에 시설이 없는 상태를 가역적으로 만들고, 전후 변화를 측정해 처분을 결정하는 절차**다. 공개자료 검색은 전수조사가 아니므로 ‘전국 최초’나 ‘유일’이라고 표현하지 않는다.

## 11. 변경 금지선

- 규제입증책임제를 새로 도입한다는 제안으로 변경하지 않는다.
- 단순 시설물 신고·철거 사업으로 축소하지 않는다.
- 주민 다수결만으로 안전·편의시설을 철거하지 않는다.
- 앱·지도·인증제 개발을 핵심으로 추가하지 않는다.
- 철거 개수나 근거 없는 절감액을 성과로 설정하지 않는다.
- 실제 확인하지 않은 시설을 불필요한 사례로 제시하지 않는다.
- 근거 없이 ‘전국 최초’, ‘유일’, ‘예산 절감’을 단정하지 않는다.

## 12. 제출 전 남은 일

| 우선순위 | 남은 일 | 처리 방법 |
| --- | --- | --- |
| P0 | 제안자 인적사항·서명 또는 전자제출 인증 | 제출자가 공식 양식 앞쪽에 입력 |
| P1 | 제안 접수 담당과 시범운영 총괄부서의 내부 지정 | 채택 후 수성구가 조직분장에 따라 지정 |
| P2 | 시설별 실제 가림·운반·보관·복원 단가 | 후보 적격성 통과 후 관리부서가 산정 |
| P2 | 실제 독자의 30초 이해 확인 | 제출 전 제목과 개요만 읽힌 뒤 취지를 재설명하게 함 |

특정 후보시설의 사전 조사·사진·소유권 확인은 제안서 완료조건이 아니다. 후보 발굴과 시설별 적격성 검증은 본 제도가 채택된 뒤의 운영 절차에 둔다.
